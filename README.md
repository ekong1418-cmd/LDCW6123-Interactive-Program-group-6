# Grab Fare Calculator System

## Course Information

**Course:** LDCW6123 – Fundamentals of Digital Competence for Programmer

**Project:** Interactive C++ Program

---

## Project Description

The Grab Fare Calculator System is a console-based C++ application developed to simulate a simple Grab e-hailing fare calculator. Users enter their name, travelling distance, and preferred Grab service. The program calculates the estimated fare based on the selected vehicle type and displays a booking summary.

This project is based on **Grab: The Evolution of Ride-Hailing Technology**, which was selected as the innovation technology for Part 1 of the assignment.

---

## Features

- Customer name input
- Travel distance input
- Grab service selection
- Fare calculation
- Booking summary
- Invalid choice detection

---

## Grab Services

- GrabBike (RM1.20/km)
- GrabCar (RM2.50/km)
- GrabPremium (RM4.00/km)
